# node-svelte-chat-app
A full-stack chat app which supports direct messages and eventually normal channels written in Svelte and Node.js.

## Technologies used
- **Front-end Side:**
  - SvelteKit and TypeScript.
- **Back-end Side:**
  - Node.js, Fastify, SQLite3.
